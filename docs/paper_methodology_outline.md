# Paper Methodology Outline

## 1. Data Collection

Three representative ORC waste heat recovery cases were selected from public literature and industry case reports:

1. Industrial kiln ORC case in Flanders, Belgium
2. Cement plant waste heat recovery case
3. Steel electric arc furnace ORC case in Germany

For each case, the study recorded available reported variables such as industry, location, ORC power output, temperature when reported, efficiency when reported, and published cost information when available.

## 2. Techno-Economic Model

For each case, annual electricity generation is estimated as:

Annual Generation = ORC Power Output × Annual Operating Hours

Annual revenue is estimated as:

Annual Revenue = Annual Generation × Electricity Price

Annual O&M cost is estimated as:

Annual O&M Cost = CAPEX × O&M Rate

Annual net benefit is estimated as:

Annual Net Benefit = Annual Revenue - Annual O&M Cost

## 3. Financial Evaluation

The project evaluates each case using:

- Net Present Value (NPV)
- Return on Investment (ROI)
- Simple Payback Period

NPV is calculated over a 15-year project life using an 8% discount rate.

## 4. Sensitivity Analysis

Sensitivity analysis tests how NPV changes when three key variables vary:

- Electricity price
- Capital cost
- Recoverable waste heat / ORC power output

The goal is to identify which business factors most affect project feasibility.

## 5. Scope and Limitations

This is not a thermodynamic design optimization study. It is a business analytics and techno-economic feasibility study. The main limitation is that not all case papers report the same level of technical detail, so some assumptions are required to make cases comparable.
