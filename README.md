# Industrial ORC Market Analysis

## Overview

This project evaluates the techno-economic feasibility and market diffusion potential of industrial Organic Rankine Cycle (ORC) systems for waste heat recovery.

## Methods

- Market sizing
- Techno-economic modeling
- ROI and NPV analysis
- Cost-benefit analysis
- Sensitivity analysis
- Bass diffusion adoption modeling
- Risk assessment

## Repository Structure

- src/: analysis code
- data/: model inputs and outputs
- figures/: project figures
- paper/: research paper

## Key Findings

Industrial ORC systems can achieve positive economic returns under favorable waste heat conditions while reducing energy losses and supporting industrial decarbonization.
