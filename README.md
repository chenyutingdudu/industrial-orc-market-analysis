# Industrial ORC Investment Analysis

This repository presents a business analytics focused study of industrial Organic Rankine Cycle (ORC) investment feasibility.

## Project Scope
The project evaluates whether an industrial firm should invest in ORC waste heat recovery equipment. It focuses on financial analysis, not thermodynamic engineering design.

## Methods
- Revenue estimation
- Return on Investment (ROI)
- Net Present Value (NPV)
- Payback period
- Scenario analysis
- Sensitivity analysis

## Repository Structure
- `paper/`: final PDF research paper
- `data/`: CSV files used in the analysis
- `figures/`: charts included in the paper
- `src/`: two simple Python files that reproduce the financial calculations and figures
- `docs/`: interview explanation for PhD applications

## How to Explain This Project
I focused on the business analytics side of an industrial energy investment problem. The main question was whether an ORC system could generate enough financial value to justify its initial capital cost. I used ROI, NPV, payback period, scenario analysis, and sensitivity analysis to evaluate investment feasibility under different market assumptions.
